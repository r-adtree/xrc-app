# Brand Opportunities · Adtree GO

Web app for TikTok GO content creators to discover and save brand campaign opportunities across Indonesia.

## Stack
- React 18
- Vite 5
- No backend required (device-based localStorage for saved jobs)

## Features
- 127 hotel brands across 34 provinces
- Filter by region, campaign type, search
- Save brands by device (no login needed)
- Adtree scoring algorithm (star rating, AOV, tier, campaign count)
- Admin panel to publish/unpublish brands

- Password: `0ADTREE1234!`
